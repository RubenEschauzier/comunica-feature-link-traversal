## Install

```bash
$ yarn add @comunica/actor-optimize-query-operation-initialize-source-link-traversal-cache
```

## Configure

After installing, this package can be added to your engine's configuration as follows:
```text
{
  "@context": [
    ...
    "https://linkedsoftwaredependencies.org/bundles/npm/@comunica/actor-optimize-query-operation-initialize-source-link-traversal-cache/^1.0.0/components/context.jsonld"  
  ],
  "actors": [
    ...
    {
      "@id": "urn:comunica:default:optimize-query-operation/actors#initialize-source-link-traversal-cache",
      "@type": "ActorOptimizeQueryOperationInitializeSourceLinkTraversalCache"
    }
  ]
}
```
