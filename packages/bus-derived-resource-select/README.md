# Comunica Bus Derived Resource Select

[![npm version](https://badge.fury.io/js/%40comunica%2Fbus-derived-resource-select.svg)](https://www.npmjs.com/package/@comunica/bus-derived-resource-select)

A comunica bus for derived-resource-select events.

This module is part of the [Comunica framework](https://github.com/comunica/comunica),
and should only be used by [developers that want to build their own query engine](https://comunica.dev/docs/modify/).

[Click here if you just want to query with Comunica](https://comunica.dev/docs/query/).

## Install

```bash
$ yarn add @comunica/bus-derived-resource-select
```
